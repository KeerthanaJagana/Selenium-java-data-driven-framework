# Selenium Java Data-Driven Automation Framework (INFO6255)

## Overview
This repository contains a Java-based Selenium test automation framework built using Maven and TestNG.
The framework follows a data-driven approach where all test data is maintained in Excel files, with no
hard-coded values in the test scripts.

The project demonstrates industry-standard QA automation practices including automatic test execution,
assertions, screenshot capture, and HTML reporting.

---

## Tech Stack
- Java
- Selenium WebDriver
- Maven
- TestNG
- Apache POI (Excel handling)

---

## Framework Highlights
- Data-Driven Automation Framework (Excel-based)
- No hard-coded test data or credentials
- Automatic sequential execution of test scenarios
- Screenshot capture before and after every test step
- Assertion-based validation using TestNG
- Custom HTML test execution report generation
- Reusable and scalable test structure
